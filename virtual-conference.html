About

Hedera Gallery Video conferencing and communication application that allows users to host private or free virtual events in Hedera Space.

Scope

This project is used by them for their NFT gallery and online .

The system have functionalities like: buying tickets with cryptocurrencies, awaiting space, gallery guidance, and others that can't be used here.

What I've done

My approach was to follow a design thinking process based on building an MVP (Minimum Viable Product).

Working on this project my role was to gather information from stakeholders, analyse the main features, to developed user flows, wireframes, mid and high-fidelity prototypes.

PROCESS

Design Process

The team assigned started by analyzing and understanding the main features to be designed and developed in phase 1. We split these features in 3 categories: nice-to-have, must and post MVP. In order to validate the idea and present it further to the users and stakeholders we decided to keep and design these features.

Results

The video conference application can support large-scale conferences with up to 400 participants, making it suitable for both exhibitions and also for large corporate events.

For the moment is still in development.
